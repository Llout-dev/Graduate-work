## О проекте

Сравнительная симуляционная система на Unity, реализующая и анализирующая два подхода к управлению поведением автономных агентов в динамической экосистеме.

## Компоненты

### GOAP-агент (готовность 95%)
- Планировщик на основе A*, пакет **CrashKonijn GOAP v3**
- Цели: `EatGoal`, `WanderGoal`
- Действия: `EatAction`, `WanderAction`
- Сенсоры: `HungerSensor`, `EatTargetSensor`, `WanderTargetSensor`

### RL-агент (готовность 60%)
- **Unity ML-Agents Toolkit** + **PyTorch**, алгоритм PPO
- Скрипт `WolfRLAgent`, пространство наблюдений: 6 параметров
- Непрерывные действия: `moveX`, `moveY`
- Статус: базовый скрипт готов, обучение блокировано несовместимостью версий ML-Agents

### Процедурная анимация (готовность 95%)
- Компонент `LegSolver2D` — аналитическое решение IK через закон косинусов
- Производительность: **~0.001 мс / вызов** при 60 FPS
- Погрешность позиционирования: < 0.05 ед.
